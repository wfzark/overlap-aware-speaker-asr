# Project Skills

These are challenge cards for future agents, not model-training skills. They are designed to help the project explore ambitious research directions without losing the baseline.

Each skill card should answer:

- What question does this skill explore?
- Why is it relevant to the current project?
- Challenge level
- Minimum viable attempt
- Stretch goal
- Failure is useful if...
- Inputs
- Outputs
- What not to do
- Success criteria
- Suggested agent prompt
- Owner suggestion

The point is not to block ideas, but to make the ideas researchable, reproducible, and clearly labeled.
