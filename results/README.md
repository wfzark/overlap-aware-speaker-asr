# Results Directory

This directory contains both curated result summaries and generated historical
records.

## Read First

- [docs/results-index.md](../docs/results-index.md)
- [figures/README.md](figures/README.md)
- [figures/curated/](figures/curated/)

## Curated vs Historical

| Type | Meaning | Location |
|---|---|---|
| Curated summaries | Reviewer-facing result summaries | `results/figures/curated/` |
| Historical wave records | Automated coordination history | `results/figures/archive/waves/` |
| Receipts | Execution/automation receipts | `results/figures/archive/receipts/` |
| Writebacks | Agent writeback records | `results/figures/archive/writebacks/` |
| Bridge checklists | Automation bridge/checklist artifacts | `results/figures/archive/bridge-checklists/` |
| Demo presentation records | Historical demo writebacks | `results/figures/archive/demo-presentations/` |
| Miscellaneous generated records | Other generated coordination/status artifacts | `results/figures/archive/miscellaneous/` |

Do not treat archived wave, receipt, writeback, or checklist files as final
benchmark claims.
